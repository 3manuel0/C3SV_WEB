### Lib3man WASM

C3SV_WEB is an experimental project where I make my C library ([C3SV](https://github.com/3manuel0/C3SV)) work in WebAssembly for learning purposes.

> [!WARNING]
> This project is experimental and may change or break at any time.

### Goals

The goal of this project is to explore how to make my C library work with wasm and hopefully create a website tha could be used to parse csv files, and do some useful stuff from it
